# 🛒 POS - Punto de Venta

## 📌 Descripción

Sistema de **Punto de Venta (POS)** desarrollado con **PHP, MySQL y Vue.js**, diseñado para gestionar operaciones comerciales de manera eficiente.

Permite registrar:

- 🧾 Ventas  
- 💳 Cuentas  
- 📦 Apartados  
- 📝 Cotizaciones  

Ideal para negocios pequeños y medianos.

---

## 🚀 Características

- Registro de ventas en tiempo real  
- Gestión de clientes y cuentas  
- Control de apartados  
- Generación de cotizaciones  
- Interfaz dinámica con Vue.js  
- Sistema rápido y fácil de usar  

---

## 🛠 Tecnologías utilizadas

- **Frontend:** Vue.js, HTML, CSS  
- **Backend:** PHP  
- **Base de datos:** MySQL
